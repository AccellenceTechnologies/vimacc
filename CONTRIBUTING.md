# Contributing to vimacc

We welcome contributions that improve the Docker setup and deployment experience for vimacc.

## How to Contribute

1. **Fork** the repository.
2. Create a **feature branch** (`git checkout -b feature/your-topic`).
3. **Commit** your changes with meaningful messages.
4. **Push** your branch to your fork.
5. Open a **pull request** (PR) against `main`.

## Code Style

- Keep Dockerfiles and shell scripts clean and readable.
- Prefer comments in English.

## Reporting Issues

If you encounter any problems, please:
- Check if an issue already exists
- If not, create a new one with clear reproduction steps

## Contact

If you're unsure whether a change is appropriate, feel free to [open an issue](https://github.com/AccellenceTechnologies/vimacc/issues) or contact us directly.
